# Mine Sweeper Game

Embark on an exciting Minesweeper adventure with the ability to tailor the game to your preferences. Customize the board size and strategically choose the number of mines for an engaging challenge. Unleash your deductive skills as you strategically flag potential mine locations. Clicking on an empty spot triggers a thrilling domino effect, revealing adjacent tiles with no mines. Immerse yourself in this classic game, blending strategy and logic, with a personalized touch.
